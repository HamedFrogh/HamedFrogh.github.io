Projects

TrainDetection Embedded ML

Lightweight ML pipeline for detecting train arrival on an STM32F103C8T6 microcontroller. Sliding-window feature engineering, int8-quantized TTA regression, and a full Jupyter pipeline from raw sensor data to firmware-ready deployment via Maple-style bootloader.

STM32 C Python TFLite int8 quant
SQIsign on Pico 2 Post-Quantum Crypto

Porting the SQIsign post-quantum signature scheme to Raspberry Pi Pico 2 (Cortex-M33). Achieved successful key generation and verification runs via bare-metal HAL, custom linker scripts, and include-path fixes across the SQIsign source tree.

Raspberry Pi Pico 2 C arduino-pico SQIsign
FL-IDS Research

Federated Learning Intrusion Detection System for heterogeneous edge IoT networks. Defends against Byzantine poisoning attacks using AutoGM and ZBDU aggregation, handles non-IID data distributions, and uses adaptive client selection.

Python PyTorch Federated Learning IoT
Credit Card Fraud Detection ML

Fraud detection on imbalanced Kaggle dataset using Random Forest, Gradient Boosting, Logistic Regression, and SVM. Extended with federated learning simulation on non-IID splits, poisoning attack simulation, and concept drift detection.

Python scikit-learn SMOTE FL simulation
